# Workflow: отключение FreeIPA-пользователя и удаление SSH-ключа

Workflow выполняет действия последовательно:

```text
Отключить пользователя FreeIPA
        ↓ только при успехе
Split списка SSH-хостов
        ↓
Удалить указанный публичный ключ из authorized_keys
```

## Импорт

1. `xyops.json`;
2. `xyops-ssh-plugin.json`;
3. `workflow-disable-user-ssh-key.json`.

## Поля запуска

- URL и CA-сертификат FreeIPA;
- логин пользователя;
- публичный SSH-ключ для удаления;
- SSH-хосты по одному `host:port` на строку.

Ключ удаляется по совпадению типа и base64-части. Комментарий ключа может отличаться.
Остальные ключи, локальный пользователь, его пароль и sudo не изменяются.

Если пользователя, файла `authorized_keys` или ключа на ноде нет, SSH-шаг
завершается успешно без изменений.
